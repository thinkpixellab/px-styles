    Description

    Produce a shade (a lighter or darker version) of a color based on the value of $shade and an optional darkest to lightest shade range.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $color

    The color to be shaded. If not provided, it will be calculated as the middle value within $shade-range.

    		Any null

    		Any 0
    $shade-range

    		Any (black,white)
